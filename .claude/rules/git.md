# Git

## Branches
- `main` - Production (immer deploybar)
- `develop` - Integration
- `feature/*`, `bugfix/*`, `hotfix/*`

## Commit-Format
```
[Typ] Kurzbeschreibung

Typen: [Feature], [Fix], [Refactor], [Docs], [Test], [Config]
```

## Ignorieren
- `node_modules/`, `target/`, `build/`, `dist/`
- `.env*`
- `.idea/`, `.vscode/`
